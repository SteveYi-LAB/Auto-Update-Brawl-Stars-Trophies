# 自動更新荒野亂鬥杯數的網頁

# 這是什麼

這是一個透過php執行的網頁
你每5秒鐘可以取得你的遊戲杯數

# 如何安裝

編輯 trophies.php 中的token

你可以在 https://developer.brawlstars.com/ 取得token
將檔案上傳至php伺服器

使用 https://yourdomain/trophies.php?tag=你的玩家標籤 來瀏覽
